Version 1.2.1

Released on: 2024-06-07

Changes

  • Taskbar is no longer in beta. Version 1 will remain free and expire on 2024-11-01.
  • Some parts of the code were re-written to improve stability.
  • New features:
    • Windows can now be grouped by application.
    • Dark mode can now be set to override the system theme.
    • Option to hide window titles and only show their icons.
    • Option to show window titles in the thumbnail view.
    • The ability to set how quickly thumbnails appear after hovering.
    • Preference for showing tabs (e.g. Finder) as individual items or not.
    • Option to hide the window's app instead of minimizing it on click.
    • Preference to no longer indicate minimized or hidden windows.
  • Fixed bugs:
    • Incorrectly detecting fullscreen windows to decide whether or not to hide Taskbar (#40).
    • Unable to drag and drop tasks when using more than one monitor (#38).
    • Discord remains in Taskbar after being closed (#51).
    • Incorrect "Taskbar" task is added to Taskbar after checking for updates (#24).

Changes
